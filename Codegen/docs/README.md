# Project Documentation Standard

This repository uses a standardized docs layout.
`Codegen/docs/` is the single source of truth for process and validation.

## Canonical Docs
- Step guides: `Codegen/steps/step01_android_base/step.md` .. `Codegen/steps/step08_settings_import_emails/step.md`
- Validation gate: `Codegen/docs/validation/checklist.md`
- Machine-readable rules: `Codegen/docs/validation/validation-rules.yaml`
- Architecture overview: `Codegen/docs/architecture.md`
- Module-wise recovery learnings: `Codegen/docs/learnings/2026-03-06-android-modulewise-recovery.md`
- Contribution workflow: `Codegen/CONTRIBUTING.md`
- Decision records: `Codegen/docs/adr/`

## Rule
- Update step content in `Codegen/steps/`.
- Update validation/architecture/meta process docs in `Codegen/docs/`.
