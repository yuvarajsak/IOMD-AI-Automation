# Step 11 Snapshots: Import Passwords

This folder contains snapshots specifically for the **Import Passwords** feature, including password manager selection, import steps, and feedback flows.

## Reference Snapshots
- `17-assistant-password-entry-with-options-reference.png`: Passwords tab entry screen with the bottom sheet for manual add and import options.
- `18-assistant-password-import-steps-reference.png`: 'How it works' instructions for importing passwords from CSV.
- `19-assistant-password-compatible-managers-reference.png`: 'Compatible Password Managers' info screen with direct `GO` actions.
- `20-assistant-password-managers-list-reference.png`: Scrollable list of password managers.
- `21-assistant-password-feedback-reference.png`: Feedback overlay shown from the password managers flow.
- `22-assistant-password-import-file-browser-reference.png`: File browser activity for selecting the CSV file.
- `23-assistant-password-add-login-blank-reference.png`: Empty manual password entry form.
- `24-assistant-password-add-login-filled-reference.png`: Manual password entry form with sample data.
- `25-assistant-password-add-login-success-reference.png`: Success confirmation after saving manual credentials.
- `26-assistant-password-saved-hidden-reference.png`: Saved password card with the password value hidden.
- `27-assistant-password-saved-visible-reference.png`: Saved password card with the password value revealed.
- `28-assistant-password-delete-confirm-reference.png`: Confirmation dialog for deleting a saved password.

## Naming Convention
Snapshots are named with numeric prefixes and descriptive labels indicating the specific screen or action they represent.
